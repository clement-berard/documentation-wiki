## [ShellFish] Documentation

Le fichier `config.fish` doit etre dans le dossier `~/.config/fish/`

### Informations

Install Fish sur MacOS

```bash
$ brew install fish
```

Setting Fish as your default shell on Mac OS X

```bash
$ echo "/usr/local/bin/fish" | sudo tee -a /etc/shells
$ chsh -s `which fish`
```
